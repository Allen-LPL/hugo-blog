# 命令介绍
docker在使用的过程中会不断的累积images，docker v1.13版本之后，提供了清理的命令
`
docker system prune
`
**docker system prune**命令可以用于清理磁盘，删除关闭的容器、无用的数据卷和网络，以及dangling镜像(即无tag的镜像)


`
docker system prune -a
`
**docker system prune** -a命令清理得更加彻底，可以将没有容器使用Docker镜像都删掉
