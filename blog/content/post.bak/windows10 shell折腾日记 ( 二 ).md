# 正文 #
## **启动zsh时会报以下错误**
    > compdef: unknown command or service: git

![1.png](https://i.loli.net/2020/04/18/4W2bCzE5L6vl91h.png)

## **解决** 
执行以下命令
``` shell
rm -f ~/.zcompdump
rm -f ~/.zcompdump-$(hostname)-5.0.6
compinit
cp ~/.zcompdump .zcompdump-$(hostname)-5.0.6
```
